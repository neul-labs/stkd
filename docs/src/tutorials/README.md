## Before You Start

Make sure you have:

1. [Installed Stack](../installation/README.md)
2. [Authenticated](../commands/auth.md) with your provider
3. A Git repository to practice with

## Quick Tips

- Start with a clean working tree
- Create branches from an up-to-date trunk
- Run `gt sync` frequently
- Keep each branch focused on one change

## Getting Help

If you get stuck:

```bash
# See available commands
gt --help

# Get help for a specific command
gt submit --help

# Check current status
gt status
```
