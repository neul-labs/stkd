# gt install-skill

Install the Claude Code skill for Stack.

## Synopsis

```bash
gt install-skill [OPTIONS]
```

## Description

Installs the Stack skill for Claude Code, enabling AI-assisted stack management directly in your editor. The skill provides contextual awareness of your stack structure and can suggest operations.

## Options

| Option | Description |
|--------|-------------|
| `--path <PATH>` | Custom installation path |
| `--force` | Overwrite existing skill installation |

## Examples

```bash
# Install the skill
gt install-skill

# Force reinstall
gt install-skill --force
```
